# AI Algorithm Lab (2024 Scheme)
Course Code: PCCAL507 | Semester: S5 B.Tech CSE (AI)

This repository contains the mandatory experimental programs for the 2024 Scheme AI Algorithm Laboratory. The lab exercises are structured systematically by core AI paradigms to aid in practical mapping and evaluation.

### Module 1: State Space Search & Optimization
* Experiment 1: Basic search strategies - 8-Puzzle Problem (*)
* Experiment 2: Variants of Hill-Climbing and Genetic Algorithms (*)
* Experiment 3: Implement A* Algorithm (*)
* Experiment 16: Implement Travelling Salesman Problem (*)

### Module 2: Adversarial Search & Games
* Experiment 4: Implement Mini-Max algorithm for game playing (Alpha-Beta pruning) (*)

### Module 3: Constraint Satisfaction Problems (CSP)
* Experiment 5: Solve Constraint Satisfaction Problems (*)
* Experiment 15: Implement local search algorithms for CSP (*)

### Module 4: Probabilistic Models & Machine Learning
* Experiment 7: Implement Naïve Bayes Models (*)

### Module 5: Knowledge Representation & Expert Systems
* Experiment 6: Implement Forward Chaining Algorithm (*)
* Experiment 12: Develop a simple expert system (*)
* Experiment 14: Implementation of Knowledge Representation schemes (*)

Note: (*) Indicates Mandatory Programs under the 2024 University Syllabus regulations.

1. Navigate to the specific module folder.
